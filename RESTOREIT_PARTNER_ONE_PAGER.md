## How It Works for Partners

| Step | What You Do |
|------|-------------|
| **1. Receive the drive** | Customer brings their corrupted drive or device |
| **2. Run the relay** | Launch the memory-only relay — no software installs on customer hardware |
| **3. Cloud recovery** | restoreit's forensic engine scans, carves, and reconstructs files |
| **4. Preview results** | Review recovered files before presenting to the customer |
| **5. Deliver** | Customer accesses restored files via secure restoreit Cloud link |

**You never touch raw data.** The relay streams encrypted sectors directly to restoreit's cloud. You deliver results via a secure link.
